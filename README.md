&#x20;**Server Health Monitor**



A Python-based system monitoring tool that tracks CPU, Memory and Disk usage in real time.



*Features*



\- Real time monitoring every 10 seconds

\- Automatic alerts when thresholds are crossed

\- Logs all reports to health\_log.txt



*Tech Stack*



\- Python

\- psutil library



&#x20;*How to Run*

pip install psutil

python health\_monitor.py
